# 코스닥 무자본 M&A 의심징후 선별도구
DART(전자공시시스템) Open API를 활용해 코스닥 상장사의 무자본 M&A 관련 이상징후를 자동으로 스크리닝하는 도구입니다.

## 프로젝트 배경
무자본 M&A 자체는 위법이 아니지만 이를 이용한 소액 투자자의 피해가 발생하는 사례가 다수 발생하고 있습니다.
공개된 공시 데이터를 자동으로 취합·분석해 추가 검토가 필요한 후보 기업을 좁혀주는 스크리닝 도구를 만들고자 했습니다.

## 변수의 선택
**충족 개수 산정 (1~5번, 0~5점)**
1. 잦은 최대주주 변경
2. 유상증자 실시
3. 전환사채 발행
4. 신규사업 추진
5. 최대주주 변경을 수반한 담보계약

**별도 확인 항목**
6. 불성실공시법인 지정 여부

## 참고자료
해당 변수들은 금융감독원 무자본 M&A 합동점검 자료의 투자주의 지표를 기반으로 선정하였으며, KBS 다큐멘터리 '코스닥 개미귀신'에서의 분석을 상당부분 참고하였습니다.



## 기술 스택

- 언어: Python 
- 주요 라이브러리: OpenDartReader, pandas, openpyxl, python-dotenv
- 데이터 소스: DART Open API (금융감독원 전자공시시스템)

## 설치 및 실행

### 1. 패키지 설치
```bash
pip install OpenDartReader pandas openpyxl python-dotenv
```

### 2. API 키 설정
프로젝트 루트에 `.env` 파일을 만들고 발급받은 DART API 키를 입력합니다.
```
DART_API_KEY=발급받은_API_키
```
> DART API 키는 https://opendart.fss.or.kr 에서 무료로 발급받을 수 있습니다.

### 3. 실행
```bash
python main.py
```
