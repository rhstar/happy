========================================
  코스닥 무자본 M&A 위험 분석 - 실행 패키지 (통합 앱)
========================================

이 폴더는 독립 실행 패키지입니다.
USB 등으로 통째로 복사해 다른 컴퓨터에서도 실행할 수 있습니다.
조회 대시보드와 네트워크 분석이 하나의 앱으로 통합되어 있습니다.

----------------------------------------
[전제 조건]
----------------------------------------
대상 컴퓨터에 파이썬(Python 3.9 이상)이 설치돼 있어야 합니다.
없다면 https://www.python.org 에서 설치하세요.
  ※ 설치 시 'Add Python to PATH' 체크 권장

----------------------------------------
[사용 순서]
----------------------------------------
1) 0_최초설치_한번만.bat  더블클릭
   - 처음 한 번만. 필요한 패키지를 설치합니다.

2) 1_앱_실행.bat  더블클릭  (포트 8501)
   - 브라우저에서 http://localhost:8501 이 열립니다.
   - 왼쪽 사이드바에서 두 기능을 전환합니다:
       · 조회 대시보드 : 종목 검색 -> 위험도, 순위, 위험 경영진
       · 네트워크 분석 : 위험기업 경영진 네트워크 탐색

----------------------------------------
[종료]
----------------------------------------
실행 시 뜨는 검은 창을 닫으면 앱이 종료됩니다.

----------------------------------------
[문제 해결]
----------------------------------------
- 'python을 찾을 수 없음':
  파이썬이 설치 안 됨 또는 PATH 미등록.
  python.org에서 재설치('Add to PATH' 체크).
- 창이 깜빡하고 사라짐:
  0_최초설치를 먼저 실행했는지 확인.
- 포트 충돌 에러:
  이미 8501 포트로 실행 중인 창을 닫고 다시 실행.
